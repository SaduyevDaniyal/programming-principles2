# TSIS3 Racer Game

Advanced racer game built with Pygame only. No assets are required.

## Features

- Main menu
- Username entry
- Dynamic traffic cars
- Road obstacles: barriers, oil spills, potholes, speed bumps
- Moving road barrier event
- Coins
- Power-ups: Nitro, Shield, Repair
- Distance meter and finish distance
- Difficulty scaling
- Game over screen
- Leaderboard saved in `leaderboard.json`
- Settings saved in `settings.json`
- Settings: sound toggle, car color, difficulty

## Run

```bash
pip install pygame
python3 main.py
```

## Controls

- Arrow keys: move car
- Collect coins and power-ups
- Avoid traffic cars and barriers
